# Brave

> Uma plataforma segura, acolhedora e empoderada para mulheres. Conhecimento, orientação e comunidade.

## 🌟 Sobre o Brave

O Brave é mais do que um site. É um espaço onde mulheres encontram força, conhecimento e apoio real. Celebramos mulheres extraordinárias que mudaram a história, oferecemos orientação segura sobre relacionamentos saudáveis, e conectamos você com recursos que importam.

### Pilares do Brave

- **Inspiração**: 102+ mulheres extraordinárias que ousaram ser corajosas
- **Apoio**: Orientação clara sobre segurança, sinais de abuso e caminhos para recuperação
- **Conhecimento**: Livros curados, artigos e palestras de especialistas
- **Comunidade**: Um espaço onde sua voz importa

## 🛠 Como Usar

### Requisitos

- Node.js 18+
- npm ou bun

### Instalação

```bash
# Clone o repositório
git clone [seu-repo]
cd bravefate

# Instale dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev

# Acesse: http://localhost:5173
```

### Compilar para Produção

```bash
npm run build
npm run preview
```

### Testes

```bash
npm run test          # Rodar testes uma vez
npm run test:watch    # Modo watch para desenvolvimento
```

## Linting

```bash
npm run lint
```


## 🎨 Tecnologias

- **Frontend**: React 18 + TypeScript
- **Build**: Vite (rápido e moderno)
- **Estilos**: Tailwind CSS
- **Componentes**: shadcn/ui + Radix UI
- **Animações**: Framer Motion
- **Roteamento**: React Router v6

## 🔐 Recursos de Segurança

- **Botão de Saída Rápida**: Redireciona com privacidade garantida
- **Contatos Importantes**: 180, 190, 188 disponíveis sempre
- **Sessão Confidencial**: Dados limpos ao sair

## 📞 Contatos de Emergência

Se você está em perigo imediato:
- **190**: Polícia (emergência)
- **180**: Violência contra mulher
- **188**: Prevenção do suicídio

**Brave não substitui serviços de emergência ou profissionais de saúde.**
